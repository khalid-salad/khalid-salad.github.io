LeetCode Is Bull****

    Khalid Hourani
  • Tags: 
  • meta

A spiteful treatment of LeetCode exercises.

LeetCode is terrible, from the way it encourages this moronic culture of whiteboarding to the outright awful solutions they post. See my page on describing algorithms.

In the interest of putting a dent in this, as well as providing a useful public good, I will write up solutions to LeetCode exercises, with technical explanations on correctness and runtime where appropriate. The solutions are in Python because I can just indent my pseudocode and it usually passes the test cases.
